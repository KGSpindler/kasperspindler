Upload amounts over time

Select one or more parties and/or specific politicians. The chart aggregates uploads by week to keep the time series readable.

Summed likes or views

Choose whether to aggregate by party or by candidate, pick the metric, and limit the time window to the last month, last 3 months, or last 6 months.

Leave empty to show the top groups in the chosen time window.
Built as a static client-side dashboard for GitHub Pages. Data file expected in the same folder as this HTML page: video_data_full.csv.
